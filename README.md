# Baozi Store API

Academic project developed for the Back-End Web Development class at UNINTER.

A simple REST API for a small Chinese bun store, built with Java and Spring Boot.

## Technologies
- Java 21
- Spring Boot
- Spring Data JPA
- MySQL

## Endpoints
| Method | URL | Description |
|--------|-----|-------------|
| POST | /produtos | Create product |
| GET | /produtos | List products |
| GET | /produtos/{id} | Get product by ID |
| DELETE | /produtos/{id} | Delete product |
| POST | /clientes | Create client |
| GET | /clientes | List clients |
| GET | /clientes/{id} | Get client by ID |
| DELETE | /clientes/{id} | Delete client |
| POST | /pedidos | Create order |
| GET | /pedidos | List orders |
| GET | /pedidos/{id} | Get order by ID |
| DELETE | /pedidos/{id} | Delete order |
